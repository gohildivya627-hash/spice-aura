Spice Aura — B2B Minimal Website (Static)

UPLOAD (simple method):
1) Download the ZIP.
2) Extract it on your computer.
3) Open your GitHub repo → Add file → Upload files
4) Drag-and-drop EVERYTHING INSIDE the extracted folder (index.html + assets folder).
   (Do not upload the ZIP itself.)

GitHub Pages:
Settings → Pages → Build and deployment → Deploy from a branch
Branch: main  Folder: / (root)

IMPORTANT (Logo):
- Replace assets/img/logo-placeholder.svg with your EXACT Spice Aura logo file.
  Use the SAME filename OR update the <img src="..."> in index.html.

Hero image:
- assets/img/hero.png is used as the hero background.
  Replace it with your preferred banner image (keep the name hero.png).
